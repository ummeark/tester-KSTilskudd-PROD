Hva er monkey-testing?

Monkey-testing simulerer en uforutsigbar bruker som klikker og skriver tilfeldig i applikasjonen – uten å følge en bestemt flyt. Formålet er å avdekke krasj, ubehandlede feil og uventet oppførsel som ikke fanges opp av ordinære tester.

Hva testes
  • 🖱️ Tilfeldige knappeklikk
  • ✏️ Ugyldig og tom skjemainput
  • 📤 Skjemaer sendt ut av kontekst
  • ⬅️ Uventet tilbake-navigasjon
  • 🔗 Tilfeldige lenker innad på siden
Hva måles
  • ⚡ Ubehandlede JavaScript-feil
  • 🖥️ console.error()-kall
  • 🌐 HTTP 4xx/5xx-svar fra API
  • 💥 Feilsider og krasj
  • 📄 Antall sider applikasjonen når
Testdata brukt
  • Tom streng og mellomrom
  • Spesialtegn (!@#$%&)
  • Norske tegn (æøå ÆØÅ)
  • Veldig lang tekst (500 tegn)
  • Ugyldig e-postadresse
